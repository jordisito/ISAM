MICROSOFT WINDOWS SERVER 2012

¿Cuál es el objetivo del curso?

El objetivo principal del Diploma SQL Server 2012 es que el participante consiga los conocimientos necesarios acorde a la estructura oficial de Microsoft que le permitan obtener la certificación:“Microsoft Certified Solutions Associate: Windows Server 2012“.

¿A quién está dirigido el curso?

El presente Diploma es adecuado para profesionales de TI que quieran profundizar sus conocimientos en la implementación y administración de Windows Server 2012. También está dirigido para:

  • Ingenieros de Sistemas.
  • Administradores de Redes.
  • Jefes y coordinadores de soporte.
  • Integradores de Sistemas.

¿Cuál es el contenido del curso?

  • Curso 20410 Instalando y Configurando Windows Server 2012.
    • Implementación y Administración de Windows Server 2012.
    • Introducción a los Servicios de Dominio Active Directory.
    • Gestión de Objetos de Servicio de Dominio Active Directory.
    • Automatizar la Administración de Servicios de Dominio de Active Directory.
    • Implementación de IPv4.
    • Implementación Dynamic Host Configuration Protocol.
    • Implementación de DNS.
    • Implementación de IPv6.
    • Implementación de Almacenamiento Local.
    • Implementación de Servicios de Impresión y Archivos.
    • Implementación de Directivas de Grupo.
    • Asegurar servidores Windows usando objetos de directiva de grupo.
    • Implementación de virtualización de servidores con Hyper-V.
  • Curso 20411 Administración de Windows Server 2012.
    • Configuración y Diagnóstico de Sistema de Nombres de Dominio.
    • Mantener Servicios de Dominio de Active Directory.
    • Gestionar Cuentas de Usuario y Servicio.
    • Implementar una Infraestructura de Política de Grupo.
    • Gestión de Escritorios de Usuario con Directiva de Grupo.
    • Configuración y Solución de Problemas de Acceso Remoto.
    • Instalación, Configuración y Resolución de Problemas de la Función de Servidor de Red.
    • Implementación de Network Access Protection.
    • Optimizar los Servicios de Archivos.
    • Configuración de Cifrado y la Auditoría Avanzada.
    • Implementación y Mantenimiento de Imágenes de Servidor.
    • Aplicación de Administración de la Actualización.
    • Monitoring Windows Server 2012.
  • Curso 20412 Configuración Avanzada de Windows Server 2012 Services.
    • Implementación de Servicios de Red Avanzada.
    • Implementación de Servicios de Archivos Avanzado.
    • Implementación de Control de Acceso Dinámico.
    • Implementación AD DS.
    • Implementación de Sitios AD DS y Replicación.
    • Implementación de Servicios de Certificados de Active Directory.
    • Implementación de Servicios de Active Directory Rights Management.
    • Implementar Active Directory Federation Services.
    • Implementar Network Load Balancing.
    • Implementar Failover Clustering.
    • Ejecución Failover Clustering con Hyper-V.
    • Implementación de Recuperación ante Desastres.
