MICROSOFT SQL SERVER 2012

¿Cuál es el objetivo del curso?

El objetivo principal del Diploma SQL Server 2012 es que el participante consiga los conocimientos necesarios acorde a la estructura oficial de Microsoft que le permitan obtener la certificación:“Microsoft Certified Solutions Associate: SQL Server 2012“.

¿A quién está dirigido el curso?

  • Administradores de Base de Datos.
  • Desarrolladores y Profesionales de Bases de Datos en Business Intelligence.
  • Administradores y Desarrolladores de Sistemas.

¿Cuál es el contenido del curso?

  • Curso 20461: Consultas con Microsoft SQL Server.
    • Fundamentos de la consulta
    • Introducción a la sentencia SELECT
    • Filtrado y clasificación de datos
    • Combinación de conjuntos
    • Agrupación y vanos
    • Consulta de datos de texto completo
    • Consulta y gestión de datos XML
    • Creación de tablas y aplicación de la integridad de los datos
    • Diseño y creación de vistas, funciones en línea,Y Sinónimos
    • Insertar, actualizar y eliminar datos
    • Otros Aspectos de Modificación de Datos
    • Implementación de transacciones, gestión de errores y Dynamic SQL
    • Diseño e implementación de rutinas T-SQL
    • Utilización de herramientas para analizar el rendimiento de las consultas
    • Implementación de índices y estadísticas
    • Descripción de cursores, conjuntos y tablas temporales
    • Comprensión de otros aspectos de la optimización
  • Curso 20462: Administración de Bases de Datos con Microsoft SQL Server.
    • Planificación e instalación de SQL Server 2012
    • Configuración y administración de instancias de SQL Server
    • Configuración de componentes de SQL Server 2012
    • Migración, importación y exportación
    • Inicios de sesión, funciones y usuarios de SQL Server
    • Seguridad de SQL Server 2012
    • Espejo y replicación
    • Clustering y AlwaysOn
    • Solución de problemas de SQL Server 2012
    • Índices y Concurrencia
    • Agente, copia de seguridad y restauración de SQL Server
    • Estudios de casos de código
  • Curso 20463: Implementando un Data Warehouse con Microsoft SQL Server. Workshop (16 horas cronológicas).
    • Diseño lógico del almacén de datos
    • Implementación de un almacén de datos
    • Creación de paquetes de SSIS
    • Diseño e implementación del flujo de control
    • Diseño e implementación del flujo de datos
    • Mejora del flujo de control
    • Mejora del flujo de datos
    • Creación de un paquete robusto y reanudable
    • Implementación de paquetes dinámicos
    • Auditoría y registro
    • Instalación de SSIS y despliegue de paquetes
    • Ejecución y seguridad de paquetes
    • Solución de problemas y ajuste de rendimiento
    • Instalación y mantenimiento de los servicios de calidad de datos
    • Implementación de los datos maestros
    • Gestión de datos maestros
    • Creación de un proyecto de calidad de datos para limpiar datos
    • SSIS y minería de datos
    • Implementación de código personalizado en paquetes de SSIS
    • Identity Mapping y De-Duplicating
